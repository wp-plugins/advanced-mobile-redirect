This plugin allows you to put in a full path URL to redirect various mobile traffic.

== Description ==

A Advanced Mobile Redirect Plugin for those who use a various URL for various mobile devices. Simply put your redirect URL and you are ready to send users to your mobile-only content.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload Plugin Order to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Advanced Mobile Redirect.
4. Check `Enable Redirect` for various devices and put in a full path URL (inluding http://) to the website you would like to redirect mobile traffic from device to.
5. Select the method of redirect you want. 301 is generally the best option.
6. Hit `Save` and you are done.
